'+' é usado para somar, ex -> 2+2 -> 4 '-' é usado para subtrair, ex -> 5-3 -> 2 '*' é usado para multiplicar, ex -> 2*5 -> 10 '/' é usado para dividir, ex -> 10/2 -> 5 '%' é usado para resto da divisão inteira, ex -> 5/2 -> 1(o que sobra da divisão) '**' é usado na potência, ex -> 5**2 (é a raiz quadrada, ou seja, 5*5) -> 25 '()' '**' '*, /, %' '+, -'
